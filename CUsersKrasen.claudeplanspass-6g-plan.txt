COMPONENT LOGIC (TrackColorPopover.jsx)
---------------------------------------

Import:
  import { isValidTrackCustomColor } from './trackColorResolver.js'

State:
  const [customInput, setCustomInput] = useState('')
  const { mode, slot, customColor } = normalizeTrackColorAssignment(track)

Effect (pre-fill input):
  useEffect(() => {
    if (mode === 'custom' && customColor) {
      setCustomInput(customColor)
    } else {
      setCustomInput('')
    }
  }, [track.id, mode, customColor])

Handlers:
  const isValidInput = customInput ? isValidTrackCustomColor(customInput) : false

  const handleCustomInputChange = (e) => {
    setCustomInput(e.target.value)
  }

  const handleCustomInputKeyDown = (e) => {
    if (e.key === 'Enter' && isValidInput) {
      onChooseCustom(customInput)
    }
  }

  const handleCustomInputBlur = () => {
    if (customInput && isValidInput) {
      onChooseCustom(customInput)
    } else if (customInput) {
      setCustomInput(mode === 'custom' ? customColor || '' : '')
    }
  }

IPC PATH
--------
User enters #FF00AA and presses Enter:
  onChooseCustom(#FF00AA)
    ↓
  TrackHeaderList: onSetTrackColor(trackId, { mode: 'custom', customColor: '#FF00AA' })
    ↓
  TimelineView.handleSetTrackColor():
    - Validates via normalizeTrackCustomColor()
    - Calls window.xleth.timeline.setTrackColor(trackId, sanitized)
    - Falls back to local state mutation if IPC unavailable
    ↓
  Engine: Persists trackColorMode='custom', trackColorCustom='#FF00AA'
    ↓
  UI: fetchTracks() refetches updated track state

COLOR RESOLUTION (resolveTrackColor, line 80-89)
-------------------------------------------------
Priority order:
1. Custom hex (if mode='custom' and valid) → always wins
2. PaletteSlot (if mode='paletteSlot' and valid) → wins over auto
3. Auto by visible index → fallback
4. TRACK_PALETTE_FALLBACK → ultimate fallback

VALIDATION RULES
----------------
Valid: #RRGGBB format (6 hex chars, case-insensitive)
  - #FF00AA ✓
  - #ff00aa ✓
  - #4CC9F0 ✓

Invalid (should show is-invalid class):
  - #FFF (too short)
  - #FF00AA00 (too long)
  - rgb(0,0,0) (wrong format)
  - GG0000 (bad hex chars)
  - #FF00 (missing chars)
  - (empty string shows no border state)
